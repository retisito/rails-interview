### Tipos de Conflictos

| Tipo | Descripci√≥n | Ejemplo |
|------|-------------|---------|
| `data_conflict` | Datos diferentes | Descripci√≥n modificada en ambos lados |
| `timestamp_conflict` | Timestamps inconsistentes | Mismo item, diferentes updated_at |
| `deletion_conflict` | Eliminaci√≥n vs modificaci√≥n | Item eliminado localmente, modificado remotamente |
| `creation_conflict` | Creaci√≥n duplicada | Item creado en ambos lados simult√°neamente |

### Estrategias de Resoluci√≥n

#### 1. Last Write Wins
```ruby

  use_local_changes
else
  use_remote_changes
end
```

#### 2. Merge Changes
```ruby

  case key
  when 'description'
    "#{local_val} (merged with: #{remote_val})"
  when 'completed'
    local_val || remote_val  # True wins
  else
    remote_val  # Default: remote wins
  end
end
```

#### 3. Auto-Resolution
```ruby

  conflict_summary.all? do |diff|
    case diff[:field]
    when 'completed'

      true  # Timestamp conflicts can be auto-resolved
    else
      false  # Other fields need manual review
    end
  end
end
```
